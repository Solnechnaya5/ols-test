Software to do warehouse layout design

The reasons to use facility layout software

When you have a warehouse with an area of ​​more than 10 thousand square meters - the question arises with the efficiency of order selection. Of course, the warehouse can employ from a few dozen to hundreds of employees. Human labor is often of the highest operating costs for an organization. Any optimization could spell tremendous savings.

Issue

This issue is partially solved by the facility map. But someone or something has to draw it and give it to the employee every time he needs to move something somewhere. Using intelligent software to create warehouse layout design may help solve this problem.

In this article, I want to describe the process of map creation for a system that automatically calculates optimal routes within a facility. This is the process of digitizing a paper project into a software. For example, consider a typical warehouse with an area of ​​27.4 thousand square meters (29.56 square feet).

Steps to creating warehouse layout software

Of course, the first thing everybody visits is Google. I also wanted to find a ready-made solution and preferably for a reasonable amount of money. As you may have guessed, that did not happen.

Existing solutions generate a 3D model or image in SVG, PNG, or something similar to an architectural drawing. And I was clearly not satisfied. I needed to create data that I can upload to the database and use in an algorithm that calculates the most optimal/efficient route.

And so we had to create everything from scratch. To better understand the topic, you need to get acquainted with the terminology.

01

Location This is the physical location within a facility, typically labeled with a name and barcode.

02

Node Conditional point on the map. A route is being built at these points, for example for loaders.

03

Edge A route in the form of a conditional line. Built between two Nodes.

“My job is to create all these objects and explain to the system what to do with them. I started my first attempts to make calculations by using Excel. In semi-automatic mode, it calculated the coordinates of each object and entered the values ​​of the attributes of the objects. Then it built a "schedule" to visually check the results of calculations. During the work, a number of shortcomings of this approach were revealed. Such as duplication of names, inaccuracy in coordinates. And most importantly - it is difficult to add new items to the map.” Aleksand M., the developer of OneLogicSoft team says.

Professional warehouse layout software

The result of the work is a Professional warehouse layout software package for uploading such maps. Of all the features of the program, we used the "Dynamic Block". This is an object with many useful parameters that can be changed. In essence, these are the blocks for each element of the room, then they build a map of the warehouse.

And then the magic of the digital map begins. We process and form INSERT SQL Statements (request to load result in a database). We load the generated data into the database. The development of map construction technology has given a number of advantages.

  • reduced card execution time. From two months to two weeks.
  • no errors related to the human factor.
  • the ability to build warehouses with unique geometric dimensions
  • set unique parameters of each Location.

Warehouse layout software improvement

But this is not the end of our work in creating warehouse layout software. After all, engineers can make changes and, if necessary, add new objects. And so an additional module was written to control data quality. It controls 19 quality parameters and automatically generates a PDF report. There are many product development plans. In the near future, We plan to add the ability to paint Location in different colors depending on their load.
